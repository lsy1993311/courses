# A Scalable Random Forest Classifier

## Requirements

+ `scikit-learn` >= 0.15.2
+ `numpy` >= 1.9.2
+ `scipy` >= 0.14.1
+ `cython` >= 0.22
+ `gcc` >= 4.9.2 or `clang` >= 3.6.0

## Usage

To build the program, first type this in your command line:

```
python setup.py build_ext --inplace
```

Then try

```
python demo.py
```
